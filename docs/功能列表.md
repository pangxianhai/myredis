# 命令

## 通用命令

#### exists key [key ...] 

  判断 key 是否存在
  
  返回存在 key 的数量，若都不存在返回 0
  
#### type key

  返回 key 的数据类型，若 key 不存在返回 none
  
#### ttl key

  返回 key 的剩余过期时间 单位为妙
  
  -2 key 不存在
  
  -1 key 无过期时间
  
#### expire key seconds
  设置 key 的过期时间 单位为妙
